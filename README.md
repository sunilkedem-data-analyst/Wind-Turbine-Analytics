# Wind Turbine Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes wind turbine performance, maintenance, faults, downtime, and energy production.

## Tools Used
- Power BI Desktop
- DAX
- Power Query
- SQL Server
- Git & GitHub

## Dashboard Features
- Executive Overview
- Turbine Performance Analysis
- Fault Analysis
- Maintenance Dashboard
- Energy Production KPIs

## KPIs
- Total Energy Generated
- Availability %
- Downtime %
- Fault Count
- Maintenance Cost
- MTTR
- MTBF

## Skills Demonstrated
- Data Modeling
- Star Schema
- DAX Measures
- Interactive Visualizations
- Drillthrough
- Tooltips
- Deployment Pipelines
